Integlight is a modern, lightweight, and developer-friendly WordPress starter theme, perfect for building highly customizable websites with ease.

== Description ==

Integlight is a clean and minimalistic WordPress starter theme designed for developers who need a solid foundation to build custom themes. It is based on Underscores and includes features such as custom backgrounds, logos, menus, featured images, threaded comments, and translation-ready functionality. With a responsive layout and accessibility-ready design, Integlight ensures an optimal user experience on all devices.

== Installation ==

1. In your WordPress admin panel, go to Appearance > Themes and click the Add New button.  
2. Click Upload Theme, then select the theme's .zip file and click Install Now.  
3. Once installed, click Activate to apply the theme to your site.  

== Frequently Asked Questions ==

= Does this theme support any plugins? =  
Yes, Integlight includes built-in support for WooCommerce and Infinite Scroll in Jetpack.  

= Is this theme customizable? =  
Absolutely! Developers can modify and extend Integlight using child themes and custom CSS.  

= Is the theme translation-ready? =  
Yes, Integlight supports translation and is compatible with multilingual plugins.
